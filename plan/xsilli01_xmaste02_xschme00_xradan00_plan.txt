Komunikační kanály
  - Discord
    - Hlavní diskuzní kanál. Byl založen skupinový server The Brave Chickens, který bude v průběhu práce sloužit
      primárně pro rychlou komunikaci s ostatními členy týmu. Každý člen musí povinně kontrolovat příspěvky alespoň
      jedenkrát denně. Předpokládá se i časté užívání skupinových hovorů.

  - GitHub
    - Hlavní pracovní kanál. V rámci sekcí Issues, Pull requests a Projects budou průběžně aktualizovány samotné
      příspěvky týkající se změn v projektu. Bude zde rovněž prováděna kontrola změn pomocí code review komentářů.
      Členové týmu musí před zahájením další práce zkontrolovat aktuální stav projektu v online repozitáři a musí
      do tří dnů reagovat na případně code review komentáře. Lze urgovat přes Discord server.

  - Týmová setkání
    - Členové týmu se budou v případě potřeby scházet přímo na fakultě, aby bylo možné lépe prodiskutovat detaily.
      Pokud to však bude možné a dostačující, budou se osobní setkání nahrazovat skupinovými hovory přes Discord,
      aby se ušetřil čas s plánováním termínů setkání (pro plánování bude primárně využíván Doodle).

  - Telefon
    - Všichni členové týmu disponují telefonními kontakty na ostatní členy. V akutních případech, kdy je lhůta pro
      kontrolu Discord serveru nedostačující, lze ostatní členy urgovat využitím telefonního hovoru.

  - Komunikace s vyučujícími
    - Dle zadání bude zprostředkována skrze vedoucího projektu (xsilli01).
